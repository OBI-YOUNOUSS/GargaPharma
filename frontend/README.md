#  Nom du Projet – PharmaPlus

##  Description


 Création d'un site web pour une boutique de ventes des medicaments en gros et details avec React + Vite

L’application est conçue avec Vite + React + TS , stylisée avec [Tailwind CSS];
utiliser les composants réutilisables pour une expérience moderne et fluide.

---

##  Fonctionnalités

-  Interface utilisateur responsive
-  Composants modulaires
-  Adaptée au mobile
